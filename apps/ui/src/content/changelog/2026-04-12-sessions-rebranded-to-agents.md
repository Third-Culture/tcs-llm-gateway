**Sessions** have been rebranded to **Agents** — giving you a clearer view of your AI coding agents and their activity.

## What Changed

The Sessions page is now the **Agents** page. Instead of tracking anonymous sessions, you now see each coding agent (Claude Code, SoulForge, OpenCode, etc.) as a first-class entity with its own cost, token, and request breakdown.

## Navigating the Agents Page

1. Open the **Agents** page from the sidebar to see all your coding agents at a glance. Each card shows the agent's total cost, request count, token usage, and last active date.

   ![Agents list in light mode](/changelog/agents-rebrand-light.png)

   ![Agents list in dark mode](/changelog/agents-rebrand-dark.png)

2. Click on any agent card to view its sessions. Each session row shows the time range, number of requests, tokens consumed, duration, and cost.

   ![Agent detail in light mode](/changelog/agents-detail-light.png)

3. Expand a session to see individual requests with response previews, model used, cache status, token counts, cost, and source.

   ![Agent detail expanded in dark mode](/changelog/agents-detail-dark.png)
